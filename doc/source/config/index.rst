Configuration Guide
===================

This section contains information for Users who wish to configure Zuul during
operation, be that setting up pipelines, adding projects, or configuring jobs
or secrets for an existing project. A high-level overview of this topic can be
found in :ref:`project-configuration`.

.. note::

   If you looking for information on configuring the Zuul service itself - for
   example, configuring how Zuul can connect to the database - refer to
   :ref:`configuration`.
